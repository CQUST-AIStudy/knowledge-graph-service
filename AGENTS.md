# AGENTS.md

## 语言

- 所有说明、计划、总结、测试结果统一使用中文。
- 代码标识符、库名、接口字段名、命令参数保留英文。

## 项目概况

- 本项目是 `knowledge-graph-service`，使用 Go 标准库 `net/http` 提供知识图谱 CRUD、校验、统计接口。
- 数据库为共享 MySQL `ptadatabase`，知识图谱表使用 `kg_` 前缀。
- Redis 仅用于图谱静态 JSON 缓存，可降级为直连 MySQL。

## 工作原则

- 修改前先阅读相关代码、配置、类型定义与调用链。
- 优先最小必要改动，复用现有实现和项目约定。
- 不修改与当前任务无关的文件，不进行无必要重构。
- 接口变更必须同步检查请求参数、响应结构、状态码、错误处理与调用方影响。

## 终端与环境

- 默认使用 `pwsh` 执行 PowerShell 命令。
- 默认使用 UTF-8 编码，注意中文路径和中文输出兼容性。
- Windows 环境下避免依赖 bash 特有行为。

## Go 约定

- Handler 保持简洁，业务逻辑放在 `graph` 或 `storage` 等业务层。
- 明确处理错误，不吞错、不静默失败。
- 不忽略返回值与错误值。
- 修改接口时同步检查请求参数、响应结构、状态码和错误处理。
- 数据库读写保持边界清晰，返回值明确，错误信息可追踪。

## 验证

- 完成修改后尽可能执行：

```powershell
go test ./...
go build ./cmd/knowledge-graph-api
```

- 未验证不得声称已修复。
- 无法验证时必须说明原因。
